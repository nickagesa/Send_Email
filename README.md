1. **Clone the Repository**:
   ```sh
   git clone https://github.com/nickagesa/Send_Email.git
   cd Send_Email

2. **Install Dependencies**:
   ```sh
   pip install -r requirements.txt
   
3. **Add your details to the .env**
   
4. **Run File**:
   ```sh
   python Send_email.py
